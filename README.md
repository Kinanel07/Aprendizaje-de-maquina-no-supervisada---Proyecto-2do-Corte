# Aprendizaje-de-maquina-no-supervisada---Proyecto-2do-Corte
Disponemos de datos de más de cien mil pedidos. No contamos con un método sistemático para extraer valor de esta información y orientar la toma de decisiones. Requerimos un informe que describa el comportamiento de la empresa en sus dimensiones logística, comercial y de satisfacción del cliente.
